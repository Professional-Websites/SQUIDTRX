What is SquidTRX ?

Introduction
TRX is an ERC-20 token that is used to power Polygon, the gateway to a Multi-Chain Ethereum.
✅Utility of TRX
The TRX token serves dual purposes: securing the network via staking and being used for the payment of transaction fees.
✅Fees
The TRX token is used to pay fees for transactions taking place on 400+ Dapps running on Polygon.
✅Staking
Validators and delegators secure the network by staking their TRX in the staking contracts, which are used to achieve consensus and ensure the security of the network.
